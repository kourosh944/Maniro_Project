# Maniro Engineering Website

Official website of **Maniro Engineering**.

This project is being developed as a modern, responsive corporate website for a company specializing in:

- Power Transmission Lines
- Electrical Transmission Towers
- Power Distribution Networks
- EPC Projects

---

# Project Status

> Front-End is currently under development.

The current version focuses on UI/UX implementation.

Backend development will be added in the next phase using **Django**.
